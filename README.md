Shinobi Alliance SMP Mod

A Fabric 1.21.10 server-side mod powering:

Village selection scroll

LuckPerms village prefixes

Advancement point system (123 real vanilla advancements)

Clan ranking + progression

Custom SMP features

Installing the Mod

Place the built mod file into your server's mods folder:

C:\YourMinecraftServer\mods\ShinobiAlliance-1.0.0.jar

Building From Source

Run:

./gradlew build

Output JAR:

build/libs/ShinobiAlliance-1.0.0.jar
