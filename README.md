# Shinobi Alliance SMP Mod

A comprehensive Fabric mod for Minecraft 1.21.10 featuring an advancement-based point system, village mechanics, war events, and custom progression.

## Setup

For setup instructions please see the [fabric documentation page](https://docs.fabricmc.net/develop/getting-started/setting-up) that relates to the IDE that you are using.

### Local Dependencies

This project requires local jar dependencies in the `/libs` folder for development:
- `forge-config-api-port-21.10.1+mc1.21.10-fabric.jar`

These files are not included in the repository. You'll need to obtain them separately and place them in the `/libs` directory before building.
